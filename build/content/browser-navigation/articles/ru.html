Класс Navigation

Основные задачи класса это:

  1. Предоставить метод навигации navigation.navigate('/path/to')
  2. При вызове метода навигации найти и исполнить обработчик маршрута.
    А его результат передать в navigation.onNavigate({ ... })

constructor(routes)

  • routes - массив маршрутов (vqua-routes)

navigation.navigate(path)

  • path - строка пути по которому ищется маршрут
  • cache - JSON строка (необязательно), содержащая ответ от обработчика маршурта.
    В этом случае его поиск и исполнение производится не будут.

navigation.onNavigate({ path, statusCode, componentName, params })

  • path - строка пути запроса
  • statusCode - числовой код статуса ответа сервера
  • componentName - строка пути компонента относительно его родительской папки
  • params - объект ответа обработчика маршрута, например: { props: {}, context: {} }

navigation.onRedirect({ path, statusCode, params })

  • path - строка пути запроса
  • statusCode - числовой код статуса ответа сервера
  • params - объект ответа обработчика маршрута, например: { props: {}, context: {} }
