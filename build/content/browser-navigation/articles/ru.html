Браузерная навигация

Как установить?

{{ install }}

Пример

{{ create-navigation }}

Описание

Функция createNavigation создает объект навигации. Его задача состоит в том чтобы:

  1. Найти маршрут по запрашиваемому пути и его обработчик
  2. Исполнить обработчик маршрута и его результат передать обработчику навигации
    А так же воспользоваться кэшированной версией результата в случае первой загрузки
  3. Хранить актуальное состояние виртуального дерева
  4. Предоставить метод для управления навигацией

Функция createNavigation(params = {}, navigationCallback = (...) => {})

params

  • routes - массив маршрутов (vqua-routes)
  • cache - строка JSON с результатом обработчика первого маршрута

navigationCallback

  • liveNodes - массив объектов виртуального vqua дерева (актуальное состояние)
  • component - объект компонента (результат ответа обработчика маршрута)
    • name - название компонента
    • props - параметры компонента
    • context - контекст компонента
  • navigate(path) - метод увправления навигацией
    • path - строка нового пути
  • callback(liveNodes) - обработчик навигации.
    Необходим для сохранения текущего состояния виртуального дерева.
    • liveNodes - массив элементов виртуального vqua дерева
