Class Navigation

The main tasks of the class are:

  1. Provide navigation method navigation.navigate('/path/to')
  2. When you call the navigation method, find and execute a route handler.
    And its result is transferred to navigation.onNavigate({ ... })

constructor(routes)

  • routes - array of routes (vqua-routes)

navigation.navigate(path)

  • path - string path by which the route is searched
  • cache - JSON string (optional) containing the response from the route handler.
    In this case, its search and execution will not be performed.

navigation.onNavigate({ path, statusCode, componentName, params })

  • path - string of query path
  • statusCode - number of server response status code
  • componentName - string of component path relative to its parent folder
  • params - object of the result route handler, for example: { props: {}, context: {} }

navigation.onRedirect({ path, statusCode, params })

  • path - string of query path
  • statusCode - number of server response status code
  • params - object of the result route handler, for example: { props: {}, context: {} }
