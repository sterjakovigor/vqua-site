Browser navigation

How to install?

{{ install }}

Example

{{ create-navigation }}

Description

The createNavigation function creates a navigation object. Its task is to:

  1. Find the route along the requested path and its handler
  2. Execute the route handler and pass its result to the navigation handler
    And also use the cached version of the result in the case of the first download
  3. Keep the current state of the virtual tree
  4. Provide a method for managing navigation

Function createNavigation(params = {}, navigationCallback = (...) => {})

params

  • routes - array of routes (vqua-routes)
  • cache - JSON string with the result of the first route handler

navigationCallback

  • liveNodes - array of virtual vqua objects (current state)
  • component - component object (the result of the route handler response)
    • name - component name string
    • props - component parameters object
    • context - component context object
  • navigate(path) - navigation control method
    • path - new path string
  • callback(liveNodes) - navigation handler.
    Required to save the current state of the virtual tree.
    • liveNodes - array of virtual vqua tree elements
