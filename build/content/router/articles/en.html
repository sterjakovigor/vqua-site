Description

A set of utilities for organizing routing.

Function route(matchers|path, action, props = {}, childs = [])

matchers|path - an array consisting of lines of paths and matcher objects | or one line of the path string

  • path - path string
  • matcher - match object
    • key - matching segment name
    • match - callback function, takes the value as an argument key. Must return true if it matches | false in case of no match.
    Example of the match object: {{ matcher }}
action - handler function
props - properties object
childs - array of nested routes

Returns the route object:

  • segments - array of path string segments
  • path - path string
  • action - request handler function
  • props - query properties object
  • childs - array of nested routes

Function separateRoutes(routes, segments = [])

Splits all nested routes to the first level

routes - array of nested routes
segments - array of segments of the current context (optional)

Returns an array of routes.

Function matchRoutes(routes, path)

Finds the first route it finds on the basis of the specified path.

routes - array of routes
path - path string

Return object routeInfo:

  • path - path string
  • segments - array of routes
  • action - route handler function
  • props - route object properties
  • childs - array of nested routes
  • request - request object
    • path - path string
    • segments - array of routes
    • params - route object properties
