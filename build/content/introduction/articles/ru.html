Вступление

Vqua это JavaScript бибилиотека для создание веб-интерфейсов, ориентированная на создания изморфных/унивесральных приложений. Или говоря более простым языком то ваш сайт будет лучше индексироваться поисковыми системами.


Экосистема

Основное

  • vqua - виртуальный DOM
  • berries - утилиты использумые в vqua

Серверный рендер

  • vqua-router - универсальный роутер
  • vqua-navigation - браузерная навигация
  • vqua-server - серверный пререндер
  • vqua-boilerplate-ssr - каркас универсального приложения
  • vqua-todo-ssr - пример заметок как на видео

Другое

  • memorux - контейнер состояний для vqua компонентов
  • dom2vqua - преобразует dom ноды в vqua ноды
  • html2vqua - преобразует html строку в vqua ноды
  • vqua2string - преобразует vqua ноды в строку
  • vqua-interpolate - преобразует переменные в текстовых нодах
