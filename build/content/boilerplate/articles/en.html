Boilerplate

Boilerplate will allow you to quickly start creating a web-interface with server-based prerendering.

How to install?

{{ install }}

Then open the browser page http://localhost:8080/ and you should see:
Welcome! Test page.

How to edit the code?

To make changes immediately after saving, open a new window and use the command:

{{ webpack }} More about webpack

How to edit css styles?

To track changes in styles in a new window, run:

{{ gulp }}

It is preferable to place styles in folders with components. The order of styles should not matter. For this you can take advantage of bem methodology.
